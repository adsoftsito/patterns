El Facade es un patrón estructural que:

provee una interfaz simple y unificada a un conjunto de subsistemas complejos.

En corto:
👉 Facade = “cara bonita” de un sistema complicado

Idea clave para memorizar

Facade simplifica, no reemplaza

| Rol           | Clase                                 |
| ------------- | ------------------------------------- |
| **Facade**    | `HomeTheaterFacade`                   |
| **Subsystem** | `DVDPlayer`, `Amplifier`, `Projector` |
| **Client**    | `Main`                                |


Diferencia con otros patrones

Facade vs Adapter

Adapter cambia interfaces

Facade simplifica uso
